## ABOUT
- SERVER ENVIRONMENT USED - NODE JS
- PACKAGE MANAGER - NPM
  - PACKAGES USED - EXPRESS, FORMIDABLE, PATH, FS, FAST-CSV, PG
- DATABASE USED - POSTGRES SQL
- MODULE USED TO TEST API - POSTMAN
- API TYPE - POST
- HOST USED - LOCALHOST

## SETUP
- INSTALL NODE
- INITIATE NPM ```npm init```
- INSTALL NPM PACKAGES
    ```npm i express```
    ```npm i formidable```
    ```npm i path```
    ```npm i fs```
    ```npm i fast-cv```
    ```npm i pg```
    ```npm i postman```

## TO RUN
```node index.js```
